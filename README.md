# Full Stack Web Development Roadmap (2026)

## 1. Computer Basics

Before coding, learn:

* How internet works
* Browser, server, database
* HTTP & HTTPS
* DNS
* Hosting
* Command Line (Terminal)
* Git & GitHub

### Learn:

* HTML basics
* CSS basics
* JavaScript basics

# 2. Frontend Development

Frontend = What users see in browser.

## HTML

Learn:

* Tags
* Forms
* Tables
* Semantic HTML
* Accessibility

## CSS

Learn:

* Flexbox
* Grid
* Responsive Design
* Animations
* Media Queries

## JavaScript

Learn:

* Variables
* Functions
* Arrays
* Objects
* DOM
* Events
* ES6
* Async/Await
* Fetch API

---

# 3. Frontend Framework

## Learn React.js

### Topics:

* Components
* Props
* State
* Hooks
* Routing

* Forms
* Context API

### Then Learn:

* Next.js
* Tailwind CSS

# 4. Backend Development
